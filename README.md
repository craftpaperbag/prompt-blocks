> **これはモック（試作版）です。** 現在の実装はUIプロトタイプであり、バックエンド連携や本番認証機能は含まれていません。

プロンプトの構成要素を「ブロック」として保存し、レゴのように組み合わせることで、効率的にプロンプトを作成できるツールです。

## コンセプト

- プロンプトのパーツをブロックとして管理
- 必要なブロックを選んで組み合わせてプロンプトを生成
- 穴埋め機能で、あとから値を入力できる

## 対象ユーザー

- ブラウザだけで使いたい方
- プロンプトを効率よく再利用したい方
- デジタルに詳しくなくても迷わず使えることを重視

## 起動方法

```bash
npm install
npm start
```
